# Project Overview

## Objective

This project documents the assessment, redesign, and modernization planning of a small business IT environment supporting:

- Office operations
- Manufacturing workflows
- File storage
- Wireless networking
- Surveillance systems
- Remote access requirements

The existing environment evolved organically over time and required modernization to improve:

- Reliability
- Security
- Scalability
- Manageability
- Operational consistency

---

## Primary Goals

### Infrastructure Modernization
- Standardize networking infrastructure
- Improve rack organization and cabling
- Create scalable hardware standards

### Network Security
- Implement VLAN segmentation
- Establish trust boundaries
- Separate guest, IoT, infrastructure, and client traffic

### Storage & Backup
- Improve NAS reliability
- Introduce redundancy planning
- Prepare for long-term backup and disaster recovery improvements

### Wireless Improvements
- Simplify SSID architecture
- Improve wireless coverage planning
- Reduce flat-network exposure

### Operational Improvements
- Improve documentation
- Standardize deployment planning
- Create maintainable infrastructure practices

---

## Key Focus Areas

| Area | Focus |
